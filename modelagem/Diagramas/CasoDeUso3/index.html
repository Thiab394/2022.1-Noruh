    Fluxo de Pesquisa e Página do Restaurante

       O diagrama de casos de uso do fluxo de pesquisa e página do restaurante pode ser encontrado abaixo:

    Casos de Uso

    Caso de Uso 1 - Fazer pesquisa por restaurantes

    UC07 Informações
    Descrição O usuário faz uma pesquisa por restaurantes
    Pré-condições Acesso à internet e ter uma conta cadastrada
    Ator
    • Usuário cadastrado
    Fluxo principal
    • O usuário acessa o app
    • O usuário entra na página de pesquisas
    • O usuário faz a pesquisa desejada
    Fluxo Alternativo (Filtros)
    • O usuário acessa o app
    • O usuário entra na página de pesquisas
    • O usuário faz a pesquisa escolhendo um dos filtros disponíveis
    Fluxo Alternativo (Pesquisa por nome)
    • O usuário acessa o app
    • O usuário entra na página de pesquisas
    • O usuário faz a pesquisa utilizando o campo de pesquisa por nome
    Pós-condições O usuário poderá entrar na página do restaurante pesquisado

    Caso de Uso 2 - Acessar página do restaurante

    UC08 Informações
    Descrição O usuário acessa a página de um restaurante
    Pré-condições Acesso à internet e ter uma conta cadastrada
    Ator
    • Usuário cadastrado
    Fluxo principal
    • O usuário acessa o app
    • O usuário entra na página do restaurante desejado
    Fluxo Alternativo (Horários)
    • O usuário acessa o app
    • O usuário entra na página do restaurante desejado
    • O usuário acessa os horários de funcionamento do restaurante
    Fluxo Alternativo (Avaliações)
    • O usuário acessa o app
    • O usuário entra na página do restaurante desejado
    • O usuário acessa as avaliações do restaurante
    Fluxo Alternativo (Mapa)
    • O usuário acessa o app
    • O usuário entra na página do restaurante desejado
    • O usuário acessa o mapa com a localização do app
    Pós-condições O usuário terá a informação desejada
